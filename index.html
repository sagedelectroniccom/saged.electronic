⚡ Electronics

🔧 Motors Section

A motor is an electrical device that converts electrical energy into mechanical energy (motion). It works by using magnetic forces to make a shaft rotate, allowing machines, fans, pumps, and many other devices to move.

💡 LED Section

An LED is an electronic component that emits light when an electric current passes through it. LEDs are energy-efficient, long-lasting, and commonly used for lighting, displays, and indicators in electronic devices.

🔌 Wires Section

Wires are thin strands of metal, usually copper or aluminum, that carry electric current from one place to another in an electrical circuit. We use wires to connect batteries to circuits, power cords for appliances, household electrical wiring, connecting electronic components on a circuit board, and charging cables for phones and laptops.

🔋 Batteries Section

A battery is a device that stores chemical energy and converts it into electrical energy to power electrical and electronic devices. We use it in remote controls, flashlights, clocks, mobile phones, laptops, toys, and electric vehicles.

🤖 Arduino

🔩 Arduino Parts

Arduino parts are the components that make up an Arduino board and allow it to receive input, process information, and control other devices. Examples of arduino parts include: Microcontroller (the "brain" of the Arduino that runs the program), USB Port (used to connect the Arduino to a computer for programming and power), Power Jack (used to supply external power), Digital Pins (used for digital input and output), Analog Pins (used to read analog signals from sensors), Reset Button (restarts the Arduino program), and Power Pins (provide power 5V, 3.3V, GND to components).

⚙️ Arduino Components

Arduino components are the electronic parts that are connected to an Arduino board to help it sense, control, and interact with the environment. Arduino components include: LED (produces light), Resistor (limits electric current), Push Button (provides user input), Buzzer (produces sound), Temperature Sensor (measures temperature), Servo Motor (provides controlled movement), and Ultrasonic Sensor (measures distance).

💻 Programming Arduino

Arduino programming is the process of writing and uploading code to an Arduino board so that it can control electronic components and perform specific tasks. Examples of arduino programming include: An Arduino program can be written to turn an LED on and off, read a sensor, or control a motor. Arduino uses a simplified version of C++ programming language that is easy to learn for beginners.
